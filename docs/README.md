# Documentação do Projeto

Documentos auxiliares para configuração e manutenção do repositório.

- **Configuração de ambiente**: variáveis de ambiente (`.env`) para cada aplicação.
- **Limpeza antes do commit**: arquivos e pastas ignorados, comandos para remover do Git o que já foi commitado por engano.

Cada aplicação (`front`, `admin-portal`, `pwa-mobile`, `backend`) possui seu próprio `README.md` na raiz da pasta com instruções de execução.
